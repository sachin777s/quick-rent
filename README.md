# QuickRent - Gadget Renting Platform

## Overview

**QuickRent** is a platform for rent your gadgets and get gadgets on rent.

## Features

- **Gadgets Listing**
- **Contact US**
- **Quick Delivery**
- **Affordable Pricing**
- **Wide Variety of Gadgets**

## Technology Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
